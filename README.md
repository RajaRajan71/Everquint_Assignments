## üìÇ Project Deep Dives

### 1. Documentation & Summarization using RAG
**Folder:** `Documentation_&_Summarization_using_RAG/`  
This project implements a Retrieval-Augmented Generation pipeline. It allows users to upload documents and get context-aware answers and summaries, reducing LLM hallucinations.
* **Key File:** `Documentation_&_Summarization_using_RAG.ipynb`

### 2. Max Profit Optimizer
**Folder:** `Max_Profit_Optimizer/`  
A solution to the "Land Development" problem using Dynamic Programming. It calculates the maximum earnings possible within a time constraint (n) by optimizing the mix of properties built.
* **Key Files:** `app.py` (Streamlit UI), `assignment1.py` (Logic)

### 3. Water Tank Problem Solver
**Folder:** `Water_Tank_Problem_Solver/`  
A interactive web application that solves the trapped rainwater problem. It features a custom SVG rendering engine to visualize how water accumulates between blocks.
* **Key Files:** `index.html`, `water_tank.js`, `style.css`
